## [2.0.0] - 2025-10-16

### 🎉 新增功能

#### RAG 系统优化
- ✨ 集成 Deepseek Embedding API，提升向量质量
- ✨ 实现本地回退机制，提高系统可用性
- ✨ 添加语义压缩功能，降低 API 成本
- ✨ 支持知识库热更新，无需重启服务
- ✨ 实现成员信息自动同步到知识库

#### 前端增强
- ✨ 优化 RAG 查询接口调用
- ✨ 改进 AI 助手对话界面
- ✨ 添加压缩提示词注入机制

#### API 扩展
- ✨ 新增 `/api/rag/refresh` - 知识库热刷新
- ✨ 新增 `/api/rag/sync-members` - 成员信息同步
- ✨ 新增 `/api/rag/status` - 系统状态查询

### 🔧 改进

#### 后端优化
- 🚀 提升 Embedding 向量维度至 1024
- 🚀 优化数据库连接池配置
- 🚀 增强错误处理和日志记录
- 🚀 改进 .env 环境变量加载机制

#### 文档完善
- 📚 创建完整的技术文档体系
- 📚 添加部署指南和最佳实践
- 📚 编写监控和日志管理指南
- 📚 提供故障排查和性能优化建议

### 🐛 修复

- 🔧 修复 Deepseek API 密钥未正确加载的问题
- 🔧 修复启动时缺少诊断输出的问题
- 🔧 修复 RAG 查询返回 500 错误的问题
- 🔧 修复热更新文件变化检测不准确的问题

### ⚠️ 破坏性更改

- 迁移后端 API 基础 URL（如有自定义集成，需更新）
